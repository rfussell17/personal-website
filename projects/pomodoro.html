POMODORO-TIMER

This is my Pomodoro Timer project that is built with HTML, CSS and vanilla JS. I built this timer to help manage my time and get the most out of studying, working on school part-time along with keeping up with my projects. No Libraries or Frameworks were used. The Timer has a Work mode (25 min) and a Rest mode (5 min) as well as stop and reset buttons.
