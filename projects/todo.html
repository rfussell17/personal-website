TO-DO APP

This is my To-Do List App project that is built with HTML, CSS and vanilla JS. You can add items to complete, cross them off when finished and clear items that have been completed. This App mainly showcases HTML and CSS for a clean aesthetic. I built this to keep track of my ever-growing weekly tasks. No libraries or frameworks were used as this was built to solidify JS concepts.
